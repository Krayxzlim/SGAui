Sistema de Gestión de Agenda Escolar

Plataforma integral para la gestión de talleres educativos

SGA centraliza la administración de talleres, colegios, usuarios y agendas en una arquitectura moderna desacoplada, construida con Spring Boot, JavaFX y MySQL.

Arquitectura técnica

Tres capas, una sola plataforma

El proyecto adopta una arquitectura desacoplada donde el backend expone una API REST que el frontend consume de forma independiente, con MySQL como capa de persistencia relacional.

🖥️

Frontend · JavaFX

Aplicación de escritorio con interfaces modulares construidas en FXML y Scene Builder. Consume la API REST con autenticación JWT.

JavaFX FXML Scene Builder CalendarFX
⚙️

Backend · Spring Boot

API REST por capas con seguridad basada en tokens JWT, control de roles y persistencia mediante JPA/Hibernate.

Spring Boot Spring Security JWT Hibernate Maven
🗄️

Datos · MySQL

Persistencia relacional con entidades JPA: Usuarios, Talleres, Colegios, Agendas y Asignaciones con sus relaciones definidas.

MySQL JPA Entidades relacionales
01
Seguridad

Autenticación y control de acceso

El acceso al sistema está protegido mediante JWT integrado con Spring Security. Cada usuario se autentica con su email y contraseña, y recibe un token con su rol que habilita únicamente las funciones que le corresponden.

  • JWT stateless — tokens con expiración configurable sin estado en servidor
  • Spring Security — protección de endpoints según rol en el backend
  • Tres roles — ADMINISTRADOR, SUPERVISOR y TALLERISTA con permisos diferenciados

02
Colegios

Gestión de instituciones educativas

Permite registrar y mantener actualizada la información de los colegios donde se dictan los talleres. Cada institución puede asociarse a múltiples talleres y eventos del calendario.

  • CRUD completo — alta, modificación, eliminación y consulta
  • Datos institucionales — nombre, dirección y contacto centralizado
  • Relación con talleres — cada colegio vincula sus actividades y agendas

03
Talleres

Administración de contenido educativo

Módulo central del sistema. Gestiona los talleres educativos con su nombre, descripción, talleristas asignados y su relación con colegios y agendas. Es el eje que conecta los demás módulos.

  • Registro completo — nombre, descripción y talleristas asignados
  • Integración con agenda — los talleres se vinculan directamente a eventos del calendario
  • Asignación de talleristas — relación entre usuarios TALLERISTA y sus actividades

04
Usuarios

Gestión de acceso y perfiles

Permite administrar el ciclo completo de usuarios del sistema. El acceso a este módulo está restringido al administrador, quien puede crear, modificar y eliminar usuarios asignándoles el rol correspondiente.

  • CRUD de usuarios — alta, modificación y baja con validación de datos
  • Roles diferenciados — ADMINISTRADOR con acceso total, SUPERVISOR y TALLERISTA con permisos acotados
  • Seguridad por JWT — el token identifica el rol y habilita los endpoints correspondientes

05
Agenda

Planificación visual de actividades

El módulo más importante del sistema. Permite visualizar y gestionar las actividades en un calendario interactivo con vistas diarias y mensuales, facilitando la planificación operativa y el seguimiento de talleres programados en cada institución.

  • CalendarFX — calendario diario y mensual con eventos dinámicos
  • Gestión de horarios — registro de eventos con hora, colegio y taller vinculado
  • Reprogramación — permite modificar y reagendar actividades existentes
  • Distribución temporal — visibilidad completa de la operación en el tiempo

Reportes

Trazabilidad y métricas del sistema

Provee visibilidad sobre el desempeño operativo: talleres dictados, colegios alcanzados, alumnos impactados y horas ejecutadas. Los datos son actualizables en tiempo real y exportables a Excel para análisis externo.

  • Exportación a Excel — generación de reportes descargables para análisis externo
  • Trazabilidad — control histórico de actividades por taller, colegio y período
